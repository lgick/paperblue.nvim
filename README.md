**paperblue.nvim** is a clean and minimal light colorscheme for Neovim, written in pure Lua.

## 📦 Installation

```lua
vim.pack.add({
  { src = 'https://github.com/lgick/paperblue.nvim' },
}
```

## 🚀 Usage

```lua
vim.cmd[[colorscheme paperblue]]
```
